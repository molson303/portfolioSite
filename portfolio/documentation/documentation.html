General information

Mustache is a creative template with unique one page design that fits best for resume, portfolio, creative personal presentation. Key features:

  • Unique one-page layout
  • Responsive design
  • Masonry gallery
  • Fixed to top navigation
  • Responsive slider
  • Bootstrap support
  • Retina screen ready
  • Contact form php files included
  • Cross browser support

Template file structure

Mustache template has following structure of the files:

Show screenshot

Images
Directory for custom images. Masonry, slider content should be placed in this directory.
  • Images
    Directory for custom images. Masonry, slider content should be placed in this directory.
  • Layout
    Contains all scipts, css and images that are required
    • Css
      Contains styles of template:
      • bootstrap.min.css
      • bootstrap-responsive.min.css
      • style.css
    • Img
      Images for template layout
    • Js
      • bootstrap.min.js
      • functions.js - contains all template functions that are called with script.js
      • jquery.js
      • script.js - This file is basic for template. Contains all function calls.
    • Plugins
      Template uses several plugins:
      • ajaxform - contact page ajax form
      • flexslider
      • isotop - masonry gallery script
      • mousewheel - gets crossbrowser wheel events
      • prettyphoto - lightbox effect is povered with this plugin
      • scrollto - page scroll animation
  • Mail
    This directory contains "contact_form.php" script that sends mail from server. Edit this file to define mail adress where message should me sent. This script requires PHP to be installed on your server.
  • index.html
    Mustache is one-page template, so it has only one html file.

Main Menu markup

Menu is based on bootstrap menu markup. It has two variant: full variant (for big screens and tablets) and minified (for smartphones). And two states: general and fixed. When screen is scrolled to menu position - it becomes fixed to top.

Menu contains small logo. It is different image from logo in the center of main page.
There are 3 logos:

  • big logo in the beginning of main page. (Line 53)
  • Logo 2 for fixed menu (Line 90) Max height 64px
  • And logo 3 for mobile devices (Line 86) Max height 60px

Pages markup

Each page should have ID (like "page1") and classname "page-item-class". Pages structure is different depending on the content.
So if you need one more page - copy code of the page that has the most suitable layout.
You can apply "scroll_to" classname to links - this will create animated transitions inside you template.

Slider details

Slider is powered with FlexSlider, uses general functionality and has no addons. So you can setup it in general way. Init scripts are in layout/js/functions.js file.

Masonry gallery

Masonry gallery is powered with Isotope. It is integrated with lightbox (PrettyPhoto). Uses horizontal layout with mouse gragging for wide screens. And vertical layout for tablets and small screens.

Contact page

Contact form uses ajax and contact_form.php file to send mails withount CMS.
To set Google map just replace it with your map code in HTML format.
